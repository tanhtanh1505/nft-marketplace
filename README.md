## NFT Marketplace

### Create project

Create project IPFS in <a href="https://app.infura.io/dashboard">here</a>

### Install requirement

Install packages

```
npm i
```

Install packages in client

```
cd client && npm i
```

### Create .env

Create .env.local like <a href="/client/.env.local.example"> .env.local.example </a> in client

### Migrate

```
truffle migrate
```

### Run

```
cd client
npm run dev
```
